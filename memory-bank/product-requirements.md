# 睡前故事生成网站 - 产品需求文档

## 一句话目标
为女朋友打造一个能自动生成/展示生动有趣睡前小故事的网页应用，摆脱俗套的动物寓言，提供温暖、惊喜、富有想象力的晚间阅读体验。

## 非目标
- 不做用户注册/登录系统（单机体验优先）
- 不做社交分享功能（专注二人世界）
- 不做长篇连载小说（控制在5-10分钟阅读时长）
- 不做复杂的后台管理系统

## 需求分类

### Must Have（必须有）
- [ ] 故事生成功能：能生成非动物主角、剧情有转折的原创小故事
- [ ] DIV小说排版：仿网络小说阅读体验，段落清晰、字号舒适、行间距适中
- [ ] 夜间模式：暖色调护眼主题，适合睡前阅读
- [ ] 故事分类/标签：按主题快速筛选（主打轻悬疑/微恐怖，辅以奇幻、爱情、科幻等）
- [ ] 一键换故事：快速生成/切换新故事
- [ ] 悬疑度控制：可调节故事的悬念强度（微恐怖到轻悬疑）

### Should Have（应该有）
- [ ] 语音朗读：支持TTS语音播放故事（方便"听故事"）
- [ ] 故事收藏：本地存储喜欢的故事
- [ ] 阅读进度记忆：记录上次阅读位置
- [ ] 故事长度控制：短篇/中篇切换（3分钟 / 8分钟）

### Could Have（可以有）
- [ ] 背景音乐：轻柔白噪音/钢琴曲
- [ ] 故事插图：AI生成故事配图
- [ ] 定时关闭：读完自动熄屏/关闭音频

## 验收标准
1. 生成一个故事用时 < 5秒
2. 故事主角非动物的比例 ≥ 90%
3. 移动端和桌面端都能正常阅读
4. 夜间模式下阅读30分钟无明显眼疲劳
